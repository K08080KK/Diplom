# Кулінарна книга
Кваліфікаційна робота на підтвердження ступеня фахового молодшого
бакалавра ([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)). Керівник
роботи – **Олійник В.В**.
## Завдання до дипломної роботи
Розроблений програмний продукт має являти собою веб сайт з генерацією кулінарних рецептів. Він має надавати всю необхідну функціональність: інтуїтивно зрозумілий інтерфейс, генерація рецепту за введеними інгредієнтами, інструкція приготування блюда, тощо. Для керування вмістом сайту використати базу даних SQLite. Розробку виконати мовами програмування JavaScript, Python Django  використовуючи API: Generative Language API(gemini-2.5-flash-lite), Custom Search API.
